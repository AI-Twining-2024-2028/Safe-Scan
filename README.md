# 🌿 Safe Scan 

An AI-powered multilingual hate comment detector with an interactive garden visualization. Watch your words bloom or wither based on their emotional impact!

## ✨ Features

- 🌸 **3-Class Classification**: LOVE, OFFENSIVE, HATE
- 🌍 **Multilingual Support**: English, Hindi, Bhojpuri
- 🎨 **Interactive Garden**: Visual plant growth based on comment sentiment
- 🌓 **Dark/Light Mode**: Toggle between themes
- 🔐 **Simple Login**: username/password works 
## 🚀 How to Run

1. Clone the repository
2. Open `index.html` in your browser
3. Enter username and password
4. Start planting comments and watch the garden grow!
